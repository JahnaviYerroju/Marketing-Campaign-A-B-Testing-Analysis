# SQL A/B Testing Analysis

## Project Overview
This project analyzes an A/B test conducted on two website variants.

Variant A: Control page
Variant B: New landing page

## Goals
- Compare conversion rates
- Calculate revenue per user
- Measure conversion lift

## Tools
SQL
Snowflake

## Key Metrics
Conversion Rate
Revenue Per User
Conversion Lift

## Analysis
SQL queries calculate user counts, conversions, revenue performance, and experiment lift.
